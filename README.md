# Interactive Wall Calendar

This is a simple, interactive calendar built with React for the web. It looks like a physical wall calendar and lets you easily click dates and write notes.

## Features

- **Wall Calendar Design:** Has a large visual image section next to the dates, just like a real calendar.
- **Select Dates:** You can click to select a start date and an end date. 
- **Add Notes:** Write notes for specific dates or ranges of dates. If you don't select any dates, you can just jot down general notes.
- **Automatic Saving:** Everything you write is saved instantly in your browser. If you refresh the page or come back later, your notes will still be there.
- **Mobile Friendly:** Works great on desktop computers, tablets, and phones.
- **Dark Mode:** Automatically changes to dark colors if your device is set to dark mode.

## Built With

- **React / JavaScript**
- **Vite** (To run the app fast)
- **Tailwind CSS** (For custom styling)
- **date-fns** (To help calculate dates easily)
- **lucide-react** (For clean icons)

## How to Run It on Your Computer

1. Make sure you have Node.js installed on your computer.
2. Open your terminal in this folder and install inside dependencies by running:
   ```bash
   npm install
   ```
3. Start up the app by running:
   ```bash
   npm run dev
   ```
4. Finally, open the link printed in your terminal (which is usually `http://localhost:5173/`) in your web browser.
